# Screenshots

This folder contains example screenshots showing the bot's functionality.

## Required Screenshots

Please save your Discord screenshots with these exact filenames:

1. **ticket-panel.png** - The main ticket panel showing all ticket reasons and the Create Ticket/Contact User buttons
2. **create-ticket-dropdown.png** - The dropdown menu when a user clicks "Create Ticket"
3. **verification-ticket.png** - A verification ticket thread showing the initial message, buttons (Close, Claim, Verify User), and verification instructions
4. **ticket-claimed.png** - A ticket after a moderator has claimed it (showing "Claimed by [username]" status)
5. **ticket-closed.png** - A ticket after being closed (showing "Closed" status with red color)

## How to Save Screenshots

1. Take screenshots in Discord (Windows: `Win + Shift + S`, Mac: `Cmd + Shift + 4`)
2. Save them to this folder with the exact filenames listed above
3. The images will automatically appear in the README when viewed on GitHub
